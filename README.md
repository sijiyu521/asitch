# AI 工作空间

> 用于规范人机协作、沉淀提示词模板和项目上下文的个人工作空间。

## 目录说明

```
ai-workspace/
├── AI工作文档.md      # 主文档：协作规范与原则
├── README.md          # 本文件：快速导航
├── prompts/           # 常用提示词模板
│   ├── 功能开发.md
│   ├── 代码审查.md
│   ├── 设计讨论.md
│   └── bug修复.md
├── configs/           # 项目上下文与 AI 设置
│   ├── project-context.md
│   └── ai-settings.md
├── outputs/           # AI 生成的中间产物
├── iterations/        # 迭代记录、复盘
└── references/        # 参考资料、链接
```

## 快速开始

1. 打开 `AI工作文档.md` 了解协作规范。
2. 根据当前任务选择 `prompts/` 下的模板。
3. 将 AI 输出保存到 `outputs/` 或 `iterations/`。
4. 定期整理 `references/` 中的资料。

## 使用建议

- 新任务开始时，先贴 `configs/project-context.md` 给 AI 建立背景。
- 频繁使用的提示词可以复制到 `prompts/` 中沉淀。
- 每次重要迭代在 `iterations/` 留档，便于回溯。
