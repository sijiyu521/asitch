# 代码审查提示词模板

## 用法

复制以下内容，粘贴代码后发送给 AI。

---

请对以下代码进行 review，重点关注：

1. **正确性**：是否有明显 bug 或逻辑错误？
2. **性能**：是否有性能瓶颈或不必要的开销？
3. **可读性**：命名、结构是否清晰？
4. **安全性**：是否有敏感信息泄露、注入等风险？
5. **与项目风格的统一性**：是否符合现有代码规范？

**代码**：

```
[在此处粘贴代码]
```

**背景说明**（可选）：
[说明这段代码的作用和上下文]

---

## 期望输出格式

- 总体评价
- 按问题严重程度列出的问题点
- 具体修改建议（可给出代码片段）
